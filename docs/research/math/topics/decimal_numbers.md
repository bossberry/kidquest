# Decimal Numbers

## Purpose

Introduce numbers smaller than one and connect decimals with money and measurement.

---

## Core Concepts

Examples:

0.5

0.25

1.75

2.3

---

## Place Value

| Value      | Example |
| ---------- | ------- |
| Ones       | 3       |
| Tenths     | 0.1     |
| Hundredths | 0.01    |

Example:

2.45

- 2 ones
- 4 tenths
- 5 hundredths

### Compare Decimals

Which is larger?

0.7 or 0.5

Answer:

0.7

---

## Common Misconceptions

### More digits means larger number

Incorrect:

0.15 > 0.8

Correct:

0.15 < 0.8
