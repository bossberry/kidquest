# Matrices

## Purpose

Help children understand how numbers can be organized into rectangular arrays, building the foundation for linear algebra, computer graphics, data science, and higher mathematics.

---

## Core Concepts

A matrix is a rectangular arrangement of numbers.

Matrices help organize and manipulate information efficiently.

Examples:

- spreadsheets
- tables
- images
- computer graphics

Matrices allow many calculations to be performed systematically.

## Learning Progression

### Stage 1: Arrays

Example:

1 2 3

4 5 6

Children recognize rows and columns.

---

### Stage 2: Matrix Notation

Example:

[1 2]

[3 4]

This is a matrix.

---

### Stage 3: Dimensions

Example:

[1 2 3]

[4 5 6]

has:

2 rows

3 columns

Dimension:

2 × 3

---

### Stage 4: Elements

Example:

[7 8]

[9 10]

Element in row 2, column 1:

9

---

### Stage 5: Matrix Addition

Matrices with the same dimensions can be added.

Example:

[1 2]

[3 4]

-

[5 6]

[7 8]

=

[6 8]

[10 12]

---

### Stage 6: Scalar Multiplication

Multiply each element by the same number.

Example:

2 ×

[1 3]

[2 4]

=

[2 6]

[4 8]

---

### Stage 7: Identity Matrix

Example:

[1 0]

[0 1]

Multiplying by the identity matrix leaves values unchanged.

---

### Stage 8: Matrix Multiplication

Matrix multiplication combines rows and columns to create new matrices.

This operation is fundamental in advanced mathematics.

---

## Rows and Columns

Rows run horizontally.

Columns run vertically.

Example:

[1 2 3]

[4 5 6]

Rows:

2

Columns:

3

---

## Dimensions

Examples:

1 × 3

2 × 2

3 × 4

Dimensions describe matrix size.

---

## Matrix Addition

Matrices must have identical dimensions.

Example:

2 × 2

-

2 × 2

Allowed.

---

2 × 2

-

3 × 2

Not allowed.

---

## Scalar Multiplication

Example:

3 ×

[2 1]

[4 5]

=

[6 3]

[12 15]

---

## Identity Matrix

Example:

[1 0]

[0 1]

Functions like the number 1 in multiplication.

## Common Misconceptions

### Rows and Columns Are Interchangeable

Rows and columns have different roles.

---

### Matrices Are Just Tables

Matrices have mathematical operations.

---

### Matrix Addition Always Works

Dimensions must match.

---

### Matrix Multiplication Is Element-by-Element Multiplication

Matrix multiplication follows different rules.
