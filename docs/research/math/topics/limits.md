# Limits

## Purpose

Help children understand how quantities behave as they approach a value, building the foundation for calculus, continuity, derivatives, integrals, and advanced mathematical analysis.

---

## Core Concepts

A limit describes what value a function approaches as the input approaches a certain number.

Limits focus on approaching a value rather than necessarily reaching it.

Limits are one of the central ideas of calculus.

## Learning Progression

### Stage 1: Approaching Values

Example:

1, 1.9, 1.99, 1.999

These numbers approach:

2

---

### Stage 2: Patterns

Example:

10, 5, 2.5, 1.25

Values approach:

0

Patterns help develop intuition.

---

### Stage 3: Functions

Example:

y = x + 1

As x approaches 3,

y approaches 4.

---

### Stage 4: Graphical Interpretation

Graphs help visualize approaching behavior.

The function may approach a value from either side.

---

### Stage 5: Left-Hand and Right-Hand Limits

Approach from:

- the left
- the right

Both sides must agree for a limit to exist.

---

### Stage 6: Infinite Behavior

Some functions increase without bound.

Example:

1, 10, 100, 1000

These values approach infinity.

---

### Stage 7: Holes and Discontinuities

A function may have a missing point but still possess a limit.

Limits describe nearby behavior.

---

### Stage 8: Limits as Foundations of Calculus

Derivatives and integrals are built upon limits.

## Common Misconceptions

### Limits Require Reaching the Value

Limits describe approaching behavior.

---

### Infinity Is a Number

Infinity represents unbounded growth.

---

### Undefined Means No Limit Exists

Some undefined points still have limits.

---

### Left and Right Limits Are Always Equal

Different sides may approach different values.
