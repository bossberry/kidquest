# Research: Thai Language

Long-form research on Thai language curriculum for KidQuest.

## Topics to Cover

- Thai phonics system (พยัญชนะ, สระ, วรรณยุกต์)
- Age-appropriate vocabulary by grade level (อนุบาล → ป.6)
- Common Thai words children learn first
- Thai sentence structure for early learners
- Sources: หลักสูตรแกนกลางการศึกษาขั้นพื้นฐาน (MOE curriculum)

## Current Content in App

- Level 1: พยัญชนะ ก-ฮ (44 consonants, match game)
- Level 2: พยัญชนะ+สระอา (24 words)
- Level 3: คำสัตว์ (20 animal words)
- Level 4: คำ 3 พยางค์ (14 three-syllable words)
- Level 5: ประโยคสั้น word-order (9 sentences)
