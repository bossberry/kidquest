# Research: English

Long-form research on English curriculum for KidQuest (Thai children learning English).

## Topics to Cover

- Phonics approach for Thai learners (differences from native English learners)
- Sight word lists (Dolch, Fry) — relevance for Thai EFL context
- CVC → CVCC → CVCe progression

- Thai MOE English curriculum requirements by grade

- Level 2: CVC Words (24 words with sound-alike distractors)
- Level 3: Sight Words fill-in-blank (16 sentences)
- Level 4: Sentence ordering (12 sentences)
