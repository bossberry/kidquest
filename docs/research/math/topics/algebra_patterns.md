# Algebra Patterns

## Purpose

Help children recognize mathematical relationships, identify rules, and develop algebraic thinking, forming the foundation for equations, functions, and advanced mathematics.

---

## Core Concepts

Algebra patterns describe how numbers or objects change according to rules.

Children learn to:

- recognize patterns
- extend sequences
- describe rules
- predict future values
- understand relationships

Algebra begins with pattern recognition.

## Learning Progression

### Stage 1: Repeating Patterns

Examples:

🔴🔵🔴🔵🔴🔵

Rule:

AB

Next:

🔴

---

### Stage 2: Growing Patterns

Examples:

1, 2, 3, 4, 5

Rule:

Add 1

---

Examples:

2, 4, 6, 8

Rule:

Add 2

---

### Stage 3: Skip Counting Patterns

Examples:

5, 10, 15, 20

Rule:

Add 5

These patterns connect arithmetic and algebra.

---

### Stage 4: Multiplicative Patterns

Examples:

2, 4, 8, 16

Rule:

Multiply by 2

---

Examples:

3, 9, 27

Rule:

Multiply by 3

---

### Stage 5: Input-Output Tables

Example:

| Input | Output |
| ----- | ------ |
| 1     | 2      |
| 2     | 4      |
| 3     | 6      |
| 4     | 8      |

Rule:

Multiply by 2

---

### Stage 6: Missing Numbers

Example:

5, 10, ?, 20

Answer:

15

---

Example:

3, ?, 9, 12

Answer:

6

---

### Stage 7: Variables

Example:

x + 3

If

x = 4

Then

4 + 3 = 7

Variables represent unknown values.

---

### Stage 8: Simple Equations

Example:

x + 5 = 9

Answer:

x = 4

---

### Stage 9: Generalizing Patterns

Example:

2, 4, 6, 8

Rule:

2 × n

Children begin expressing rules symbolically.

Examples:

### Shape Patterns

Example:

4, 8, 12, 16

Can be represented by:

4 × n

## Relationship with Tables

Example:

| Input | Output |
| ----- | ------ |
| 1     | 3      |
| 2     | 6      |
| 3     | 9      |

Rule:

Multiply by 3

Tables organize relationships.

---

## Relationship with Functions

Every input corresponds to an output.

Example:

x → x + 2

Functions are built upon pattern recognition.

---

## Common Misconceptions

### Patterns Always Repeat

Some patterns grow instead of repeating.

---

### Every Pattern Adds the Same Number

Some patterns involve multiplication.

---

### Variables Are Objects

Variables represent numbers.

---

### Guessing Instead of Identifying Rules

Children should explain the pattern.

## Teacher Notes

Encourage explanation of rules.

Focus on relationships instead of memorization.

Use concrete examples before introducing symbols.

## Parent Notes
