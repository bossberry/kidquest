# Functions

## Purpose

Help children understand relationships between inputs and outputs, building the foundation for algebra, graphing, coordinate planes, and advanced mathematics.

---

## Core Concepts

A function is a rule that assigns exactly one output to each input.

Examples:

Input:

1

Output:

2

Input:

2

Output:

4

Input:

3

Output:

6

Rule:

Multiply by 2

Functions describe relationships between quantities.

## Learning Progression

### Stage 1: Input-Output Machines

Example:

Input:

3

Rule:

+2

Output:

5

Children think of functions as machines.

---

### Stage 2: Finding Rules

Example:

| Input | Output |
| ----- | ------ |
| 1     | 4      |
| 2     | 5      |
| 3     | 6      |

Rule:

Add 3

---

### Stage 3: Completing Tables

Example:

| Input | Output |
| ----- | ------ |
| 1     | 2      |
| 2     | 4      |
| 3     | ?      |
| 4     | 8      |

Answer:

6

---

### Stage 4: Variables

Example:

y = x + 2

If:

x = 5

Then:

y = 7

Variables represent changing quantities.

---

### Stage 5: Multiple Inputs

Example:

| x   | y   |
| --- | --- |
| 1   | 3   |
| 2   | 6   |
| 3   | 9   |

Rule:

y = 3x

---

### Stage 6: Ordered Pairs

Example:

(1,2)

(2,4)

(3,6)

Ordered pairs connect functions to coordinate planes.

---

### Stage 7: Graphing Functions

Points:

(1,2)

(2,4)

(3,6)

can be plotted on a coordinate plane.

Graphs reveal patterns visually.

---

### Stage 8: Function Rules

Examples:

y = x + 1

y = 2x

y = 5x + 3

Functions describe relationships using equations.

## Common Misconceptions

### Inputs Can Produce Multiple Outputs

In a function, each input has only one output.

---

### Functions Must Be Linear

Functions can follow many different rules.

---

### Tables and Functions Are Different Concepts

Tables are one representation of functions.

---

### Variables Represent Fixed Numbers

Variables can change.

## Teacher Notes

Begin with patterns and machines.

Avoid formal notation too early.

Focus on relationships and meaning.

## Parent Notes

Use everyday examples.

Examples:

"If one toy costs 5 dollars, how much do 3 toys cost?"

Ask:

"What is the rule?"

"What happens if the input changes?"
