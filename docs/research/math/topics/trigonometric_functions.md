# Trigonometric Functions

## Purpose

Help children understand sine, cosine, and tangent as functions that describe periodic relationships, building the foundation for waves, physics, engineering, and calculus.

---

## Core Concepts

Trigonometric functions relate angles to numerical values.

The three primary trigonometric functions are:

- sine
- cosine
- tangent

These functions are periodic and repeat in predictable patterns.

## Learning Progression

### Stage 1: Review of the Unit Circle

Every angle corresponds to a point on the unit circle.

Coordinates:

(cos θ, sin θ)

---

### Stage 2: Sine Function

Sine represents the vertical coordinate.

Example:

sin(0°) = 0

sin(90°) = 1

sin(180°) = 0

---

### Stage 3: Cosine Function

Cosine represents the horizontal coordinate.

Example:

cos(0°) = 1

cos(90°) = 0

cos(180°) = -1

---

### Stage 4: Tangent Function

Tangent is defined as:

sin θ / cos θ

Example:

tan(45°) = 1

---

### Stage 5: Graphing Sine

The sine graph forms a smooth repeating wave.

It oscillates between:

-1 and 1

---

### Stage 6: Graphing Cosine

Cosine also forms a wave.

It begins at:

1

when θ = 0

---

### Stage 7: Periodicity

Sine and cosine repeat every:

360°

or

2π radians

---

### Stage 8: Amplitude and Period

Amplitude describes height.

Period describes how long it takes for a cycle to repeat.

## Sine Function

Properties:

- range from -1 to 1
- periodic
- smooth curve

Applications:

- sound waves
- tides
- seasons

---

## Cosine Function

Properties:

- range from -1 to 1
- periodic
- phase shifted from sine

Applications:

- circular motion
- vibrations

---

## Tangent Function

Properties:

- unbounded
- periodic
- contains vertical asymptotes

Applications:

- slope
- angles
- navigation

## Common Misconceptions

### Trigonometric Functions Only Apply to Triangles

They describe many periodic phenomena.

---

### Sine and Cosine Are Linear

They produce curved graphs.

---

### Tangent Is Always Defined

Tangent is undefined when cosine equals zero.

---

### Waves Never Repeat Exactly

Trigonometric functions are periodic.
